GIRL #2 (32x32)
2020.03.15
Aseprite v1.2.25 (macOS)
And another experiment with 32x32 pixel limit. This time I used the 1/3 ratio for head/body/legs, which allows more area for facial expressions and hair, and shaded outlines also help create a more elgant shape.
DEKU #2 (32x32)
2020.03.15
Aseprite v1.2.25 (macOS)
Another experiment, with Deku, now full body but also reduction to 32x32 pixels.

I used the same ratio for body parts as the riginal anime. The result is satisfactory, but there is very little space for the face and hair.
